# Print Detail Font Size Design

วันที่: 16 กรกฎาคม 2569

## เป้าหมาย

ทำให้ข้อความรายละเอียดในเอกสาร Timeline ที่สั่งพิมพ์อ่านง่ายขึ้น โดยไม่เปลี่ยนขนาดหัวข้อ ชื่อขั้นตอน หรือการแสดงผลบนหน้าจอปกติ

## แนวทาง

ปรับเฉพาะกฎ CSS ภายใน `@media print` ของรายละเอียดที่เป็นข้อความรอง ได้แก่ subtitle และข้อความประกอบในแต่ละขั้นตอน ให้ใหญ่ขึ้นจากขนาดปัจจุบันประมาณ 16px เป็น 18px พร้อมคงหัวข้อหลักและหัวตารางไว้เท่าเดิม

## ขอบเขต

- แก้เฉพาะ `app/globals.css`
- ไม่เปลี่ยนขนาด `.print-header h1`, ชื่อขั้นตอน `.text-lg` หรือหัวตาราง
- ไม่เปลี่ยนโครงสร้างตาราง กระดาษ A4 แนวนอน หรือข้อมูล Timeline
- ตรวจสอบด้วย lint และ test ที่เกี่ยวข้อง พร้อมรัน local เพื่อยืนยันว่าแอปเปิดได้
